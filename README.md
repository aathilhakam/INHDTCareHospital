# 🏥 Hospital Management System (MERN Stack)

A full-stack **Hospital Management System** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This project is designed to streamline hospital operations such as patient management, appointments, medical records, and administrative workflows.

---

## 🚀 Features

- 👨‍⚕️ Patient Registration & Management  
- 📅 Appointment Scheduling System  
- 🏥 Doctor & Staff Management  
- 📊 Medical Records Handling  
- 🔐 User Authentication & Authorization  
- 📱 Responsive User Interface  
- 🗂️ Admin Dashboard for management operations  

---

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/aathilhakam/INHDTCareHospital.git
2. Install dependencies
Backend
cd server
npm install

Frontend
cd client
npm install
3. Configure Environment Variables

Create a .env file in the server folder and add:

MONGO_URI=your_mongodb_connection_string

4. Run the project
Start Backend
cd server
npm start
Start Frontend
cd client
npm start

🌐 API Endpoints (Sample)
POST /api/auth/register – Register user
POST /api/auth/login – Login user
GET /api/patients – Get all patients
POST /api/appointments – Create appointment
